My Library App

This page displays the movie library app. it has images of both the frontend and the backend. This app was made for the user to add his/her own movies, creating a library of their favorite movies, which they can later revisit, add more movies, or delete old ones.

Click on the images to get a larger view!

These two images show the frontend of the application, which is what the user would be using. This GUI is programmed using PyQt5

These two images show the backend of the application, which is programmed using Flask.
